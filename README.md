# KHTMX

A Library to extend the kotlinx.html dsl with binding for htmx.
It also provides a convenient dsl to build complex htmx expression for triggers.

## Roadmap

- [ ] Publishing to Maven
- [x] Trigger Dsl
- [ ] Swap Dsl
- [ ] Sync Dsl
- [ ] Configuration Dsl
- [ ] Extensions for CSRF Tokens
- [ ] Spring Autoconfiguration
- [ ] KTOR Plugin
